変数について

変数とは？

変数は、プログラム内でデータを保存するための名前付きの「箱」です。数値や文字列、リストなど、さまざまなデータを保存できます。

age = 25  # 変数ageに数値25を格納

変数のデータ型

変数にはさまざまなデータ型があります。例えば、整数や文字列、リストなどがあります。

  • 整数型（int）：数値
  • 文字列型（str）：文字や文字列
  • リスト型（list）：複数の値を保存できる

name = "Alice"  # 文字列型
age = 25        # 整数型
fruits = ["apple", "banana", "orange"]  # リスト型

変数の使い方

変数を使うことで、データを簡単に操作できます。例えば、数値を加算したり、文字列を結合したりできます。


name = "Alice"
greeting = "Hello, " + name + "!"  # 文字列の結合
print(greeting)  # 出力: Hello, Alice!
