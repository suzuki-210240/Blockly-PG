ループについて

ループとは？

ループは、同じ処理を繰り返し行いたい時に使用します。Pythonには`for`文や`while`文があります。

for文の使い方

`for`文を使うと、指定した回数だけ繰り返し処理を行います。


for i in range(5):
  print(i)  # 0から4までの数字を出力

while文の使い方

`while`文は、指定した条件が真である間、処理を繰り返します。


x = 0
while x < 5:
  print(x)
  x += 1  # xを1ずつ増加

ループの応用例

ループを使うと、リストの全ての要素を一度に処理することができます。


fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
  print(fruit)  # リストの全要素を順番に出力
