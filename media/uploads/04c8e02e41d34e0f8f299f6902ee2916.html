条件分岐について

条件分岐とは？

条件分岐は、特定の条件に基づいて、異なる処理を実行するための方法です。`if`文を使用します。


x = 10
if x > 5:
  print("xは5より大きい")  # 条件が真の場合
else:
  print("xは5以下")  # 条件が偽の場合

複数の条件分岐

`elif`を使うと、複数の条件を順番にチェックできます。


x = 10
if x > 10:
  print("xは10より大きい")
elif x == 10:
  print("xは10と等しい")
else:
  print("xは10より小さい")

条件分岐の使いどころ

条件分岐は、ユーザー入力の判定や、プログラムの状態に応じた処理の切り替えに役立ちます。


age = 18
if age >= 20:
  print("成人です")
else:
  print("未成年です")
